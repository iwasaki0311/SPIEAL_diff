### このツールの使い方

1. `index.html` をブラウザで直接開いてください。

2. 下記手順でエクスポート・解凍した「対象フォルダ」を、  
   ツール画面の「フォルダを選択」ボタンから選択してください。

```text
+-------------------------------+
|          ブラウザ             |
|-------------------------------|
|   [ index.html を開く ]       |
|                               |
|                +-----------+  |
|                | フォルダ   |  |
|                | を選択     |  |
|                +-----------+  |
+-------------------------------+
                |
                v
[SPIRALからエクスポート・解凍したフォルダを選択]
                |
                v
        [解析・ツール実行]
```

- フォルダを選択すると、自動的に解析が始まり、リソース内容が表示されます。

---

### チェック対象のリソース準備手順

このツールに正確な情報を読み込ませるには、あらかじめSPIRALで対象リソースをアプリ化しておく必要があります。

[SPIRALアプリ化手順ガイド](https://sites.google.com/gon-dola.com/solution/%E3%83%9B%E3%83%BC%E3%83%A0/5-%E8%A3%BD%E9%80%A0/git%E6%BA%96%E5%82%99)

- DBとそれに紐づくWEB機能の他、カスタムモジュール、カスタムプログラムを使用している場合は「開発」タブに設置してください。

---

1. SPIRALにログインする。
2. アプリ一覧から、対象リソースを含むアプリを選択し、表示する。
    - URLのパラメータに「_act=EditDetail」「_spiral_app_id=*****」が表示されているはずです。
3. 上記URLのうち「_act=EditDetail」の部分を「_act=Export」に書き換え、エンターキーで読み込み直してください。
    - ZIPファイルがダウンロードできます。
4. 解凍したフォルダを、このツールの読み込み対象フォルダとしてください。

```text
[SPIRALログイン]
      |
      v
[アプリ一覧表示]
      |
      v
[対象アプリ選択]
      |
      v
[URL確認]
  (例: ...?_act=EditDetail&_spiral_app_id=*****)
      |
      v
[URL書き換え]
  (→ ...?_act=Export...)
      |
      v
[ZIPダウンロード]
      |
      v
[フォルダ解凍]
      |
      v
[ツールに読み込ませる]
```

---

> 💡 **Tips**:  
> 本ツールはインターネットにデータを送信しません。ローカルのみで動作しますのでご安心ください。
